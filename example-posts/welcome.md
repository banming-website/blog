# 欢迎来到 Banming 博客

大家好！欢迎来到我的个人博客。

## 关于这个博客

这是一个基于 Next.js 构建的现代化博客，采用了以下技术栈：

- **Next.js 16** - React 框架
- **TypeScript** - 类型安全
- **Tailwind CSS** - 样式框架
- **Zustand** - 状态管理
- **Obsidian** - 内容管理

## 内容管理方式

这个博客采用独特的内容管理方式：

1. 在 **Obsidian** 中使用 Markdown 编写文章
2. 推送到 **Gitee** 仓库
3. Webhook 自动触发更新
4. 内容缓存到 **Redis**
5. 前端自动展示

## 特色功能

- 🎵 全局音乐播放器（网易云音乐集成）

- 🌙 深色模式支持
